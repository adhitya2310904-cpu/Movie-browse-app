# Movie Search App

A simple Movie Search Application built using HTML, CSS, and JavaScript that fetches movie details using the OMDb API.

## Features

* Search movies by title
* Displays movie poster
* Displays release year
* Displays genre
* Displays IMDb rating
* Displays movie plot
* Handles invalid movie searches

## Technologies Used

* HTML
* CSS
* JavaScript
* OMDb API

## Getting Started

### 1. Clone the Repository

```bash
git clone <repository-url>
```

### 2. Navigate to the Project Folder

```bash
cd Movie-Search-App
```

### 3. Get an API Key

1. Request a free API key from OMDb.
2. Activate the key through the email received.
3. Replace:

```javascript
const apiKey = "YOUR_API_KEY";
```

with your API key.

### 4. Run the Project

Open `index.html` in your browser.
